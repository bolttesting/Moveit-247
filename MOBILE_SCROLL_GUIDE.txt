┌─────────────────────────────────────────────────────────────────┐
│                  TECHNICAL SPECIFICATIONS                       │
└─────────────────────────────────────────────────────────────────┘

   Dimensions:
   • Table minimum width:  800px
   • Mobile viewport:      320px - 480px
   • Scrollable distance:  ~480px (varies)
   
   Touch Targets:
   • Buttons:             44px × 32px (iOS standard)
   • Table rows:          44px height
   • Checkbox:            24px × 24px
   
   Typography:
   • Table text:          13px
   • Button text:         10-11px
   • Hint text:           12px
   
   Colors:
   • Arrow:               #3b82f6 (blue)
   • Hint text:           #64748b (gray)
   • Hint background:     #f8fafc (light gray)
   • Scrollbar track:     #f1f5f9
   • Scrollbar thumb:     #cbd5e1
   
   Animations:
   • Arrow slide:         2s infinite
   • Hint pulse:          2s infinite
   • Scroll:              Native smooth
